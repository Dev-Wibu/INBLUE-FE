📡 API Reference – Luồng Book Slot

GET
/api/kiosks
→ Lấy list kiosk đang active để render BƯỚC 1
GET
/api/kiosks/{kioskId}/slots?date=YYYY-MM-DD
→ Lấy slot trống theo ngày. FE gọi 7 lần (mỗi ngày trong tuần) để render calendar.
Response: [ { startTime, endTime, available: true/false } ]
POST
/api/mentor-bookings/pick-slot
→ Đặt slot đã chọn.
Body: { applicationDetailId, kioskId, scheduledStart, scheduledEnd }
⚠ Nếu 409 → slot vừa bị người khác book, báo user chọn lại.
📝 Lưu ý triển khai cho FE
  • Gọi /slots?date= với từng ngày trong tuần (7 request), hoặc BE mở rộng hỗ trợ query theo range ngày để tối ưu.
  • Slot thời gian = lấy từ startTime → endTime trong response, không tự tính lại ở FE.
  • Ô available: false → disable click, render mờ.
  • Khi user click slot → lưu tạm { kioskId, scheduledStart, scheduledEnd } vào state, hiển thị confirm panel.
  • Nếu POST trả về 409 → toast "Slot vừa được đặt, vui lòng chọn lại" và refresh lịch.
  • Cần truyền applicationDetailId vào body POST – lấy từ API current-round của ứng viên.
